| 항목 | 내용 |
|------|------|
| 게임 컨셉 | 가상 주식 시장에서 매매 + 카드 심리전으로 최고 자산을 달성하는 멀티플레이 보드게임 |
| 한 줄 설명 | 숨겨진 가격 흐름과 카드 심리전으로 승부하는 주식 투자 게임 |

## 2. 핵심 컨셉

### 숨겨진 시나리오 방식
- 게임 시작 시, 각 회사의 **전체 라운드 가격 변동이 미리 결정**된다.
- 플레이어들은 이 흐름을 **알 수 없는 상태**에서 매매를 진행한다.
- 매 라운드 종료 시 차트가 공개되며, 다음 흐름을 예측해야 한다.

### 카드 심리전
- **특수 카드**: 특정 회사의 수익률을 몰래 조작. 누가 어디에 썼는지 비공개.
- **정보 카드**: 회사 정보, 타 플레이어 동향 등을 엿볼 수 있는 첩보 카드.
- **라운드 카드**: 매 라운드 전체 회사에 영향을 주는 이벤트로 자동 등장.

### 순위 긴장감 유지
- 게임 중 **등수만 공개**, 자산 총액은 비공개
- 순위표는 **Tab 키를 누르는 동안만 표시** (홀드형, 떼면 사라짐)
- 1등과의 격차를 알 수 없어 마지막까지 긴장감 유지

## 3. 게임 규칙

### 3.1 기본 구조

| 항목 | 기본값 | 방 생성 시 변경 가능 여부 |
|------|--------|--------------------------|
| 시작 자금 | 100만원 | 검토 중 |
| 회사 수 | 인원수 기반 자동 | ✅ 자동 or 방장 수동 선택 |
| 라운드 수 | 10라운드 | ✅ 가능 |
| 라운드 제한 시간 | 3분 | ✅ 가능 |

- **승리 조건**: 최종 라운드 종료 후 보유 현금 + 주식 평가액 합계 1위

### 3.2 회사 수 설계

- **최소 4개** / 플레이어 수보다 적게 유지
- 방 생성 시 두 가지 모드 선택:

| 모드 | 설명 |
|------|------|
| 자동 (유동) | 인원수에 따라 자동 결정 |
| 수동 | 방장이 직접 선택 (최소 4개 ~ 인원수-1개) |

**자동 모드 기준 (확정 전)**

| 인원 수 | 회사 수 |
|---------|---------|
| 2~3명 | 4개 |
| 4~5명 | 5개 |
| 6~7명 | 6개 |
| 8명 이상 | 7개 |

### 3.3 라운드 진행 순서

```
1. [카드 드래프트]    3분 매매 시간 내 포함 — 2장 공개 → 1장 선택 보관 / 1장 소멸 (2라운드부터)
2. [매매 페이즈]      3분 제한 — 매매 + 정보 카드 사용(라운드당 1장 제한) + 특수 카드 지정
3. [카드 연출]        특수 카드 순차 공개 애니메이션 (회사별 수익률 실시간 변동)
4. [라운드 카드]      전체 회사 일괄 적용 이벤트 공개 (균등 확률)
5. [최종 수익률]      모든 효과 반영 후 최종 등락률 확정 표시
6. [순위 공개]        등수만 표시 (자산액 비공개)
7. [보너스 카드]      3라운드마다 등수 기반 추가 카드 지급
8. [다음 라운드]      반복
```

### 3.4 매매 규칙

| 항목 | 내용 |
|------|------|
| 매수 | 현재 가격 × 수량만큼 현금 차감, 주식 추가 |
| 매도 | 보유 수량 이하로만 가능, 현금 증가 |
| 공매도 | 미지원 (v1) |
| 신용거래 | 미지원 (v1) |
| 매매 단위 | 1주 단위 |
| 제한 시간 | 3분 (방 생성 시 변경 가능) |

### 3.5 주가 변동 설계

- 게임 시작 시 랜덤 시드로 전체 라운드 가격 곡선 사전 생성
- 지나간 라운드의 가격만 차트로 확인 가능, 이후는 숨겨져 있음

**변동폭 설계**

| 구분 | 변동폭 | 등장 빈도 |
|------|--------|----------|
| 일반 구간 | ±5~15% | 대부분의 라운드 |
| 도파민 구간 | ±30~50% 이상 | 가끔 랜덤 등장 |

**회사 유형**

| 유형 | 설명 | 도파민 빈도 |
|------|------|------------|
| 안정형 | 변동성 낮고 완만한 성장 | 낮음 |
| 성장형 | 초반 낮고 후반 급등 가능 | 중간 |
| 하락형 | 전반적으로 우하향 트렌드 | 낮음 |
| 롤러코스터형 | 큰 폭의 상하 반복 | 높음 |
| 반전형 | 초반 하락 후 후반 급반등 | 중간 |

## 4. 카드 시스템

### 4.1 카드 종류 분류

| 분류 | 설명 | 사용 시점 | 라운드당 제한 |
|------|------|----------|--------------|
| 특수 카드 | 특정 회사의 수익률 조작 (비공개) | 매매 페이즈 중 | 기본 2장 |
| 정보 카드 | 회사/플레이어 정보 열람 | 매매 페이즈 중 언제든 | 1장 |
| 라운드 카드 | 전체 회사 일괄 이벤트 (자동 등장) | 라운드 결과 시 자동 | — |

---

### 4.2 특수 카드

#### 게임 시작 시 기본 지급 (전원 동일, 8장)

| 카드명 | 효과 | 수량 |
|--------|------|------|
| 급등 | 해당 회사 이번 라운드 수익률 +20%p | 3장 |
| 급락 | 해당 회사 이번 라운드 수익률 -20%p | 2장 |
| 폭등 | 해당 회사 이번 라운드 수익률 ×2 | 1장 |
| 폭락 | 해당 회사 이번 라운드 수익률 ×0.5 | 1장 |
| 반전 | 해당 회사 이번 라운드 수익률 부호 반전 (+ ↔ −) | 1장 |
| **합계** | | **8장** |

#### 사용 규칙
- 매 라운드 기본 **2장** 사용 가능 (특급 카드로 3장까지 확장 가능)
- **1번째 카드**: 어느 회사에나 자유롭게 사용
- **2번째 카드**: 반드시 1번째와 **다른 회사**에만 사용
- **3번째 카드** (특급 카드 사용 시): 반드시 앞서 쓴 회사와 **다른 회사**에만 사용
- 즉 한 라운드에 **같은 회사에 2장 이상 집중 불가** (본인 기준)
- 사용 후 소멸, 누가 어느 회사에 썼는지 **비공개**
- 다른 플레이어의 카드는 같은 회사에 중첩 적용 가능

#### 드래프트 / 보너스 풀 추가 특수 카드

기본 지급 카드 외에 드래프트와 보너스 카드 풀에는 아래 카드도 포함:

| 카드명 | 효과 |
|--------|------|
| 소급등 | 해당 회사 이번 라운드 수익률 +10%p |
| 소급락 | 해당 회사 이번 라운드 수익률 -10%p |
| 급등 | +20%p |
| 급락 | -20%p |
| 폭등 | ×2 |
| 폭락 | ×0.5 |
| 반전 | 부호 반전 |

---

### 4.3 정보 카드

- 매매 페이즈 중 **언제든지** 사용 가능
- **라운드당 1장 제한** (특급 카드 사용 시 2장으로 확장)
- 사용 즉시 효과 발동 후 소멸
- 결과는 **사용한 본인에게만** 표시

| 카드명 | 효과 |
|--------|------|
| 등락 예보 | 내가 선택한 회사의 이번 라운드 순수 등락 방향 공개 (▲/▼, 특수카드 효과 제외) |
| 카드 탐지 | 내가 선택한 회사에 현재 사용된 특수 카드 총 장수 공개 |
| 미행 | 내가 선택한 플레이어가 이번 라운드 투자한 회사명 목록 공개 (금액 비공개) |
| 내부고발 | 이번 라운드 특수 카드가 가장 많이 쌓인 회사 1곳 공개 |
| 시장 스캔 | 전체 회사 중 이번 라운드 순수 등락 상위 2개 / 하위 2개 회사명만 공개 |
| 포트폴리오 스캔 | 내가 선택한 플레이어의 현재 보유 종목 목록 공개 (수량/금액 비공개) |
| 특급 카드 | 이번 라운드 특수 카드를 1장 추가 사용 가능 + 정보 카드도 1장 추가 사용 가능 |

> 정보 카드는 드래프트 및 3라운드 보너스 카드 풀에 포함되어 랜덤 획득 가능
> 기본 지급 카드에는 포함되지 않음

---

### 4.4 카드 드래프트 (2라운드부터 매 라운드)

- 3분 매매 시간 **내에 포함** (별도 시간 없음)
- 라운드 시작과 동시에 **2장 공개** → 플레이어가 **1장 선택 보관**, **1장 소멸**
- 드래프트 풀: 특수 카드 + 정보 카드 혼합
- 선택 전까지 매매 및 카드 사용은 자유롭게 진행 가능
- 드래프트를 통해 플레이어마다 카드 덱이 점점 달라져 개성 있는 전략 가능

---

### 4.5 3라운드마다 보너스 카드 지급

- 3, 6, 9라운드 결과 발표 후 등수 기반 차등 지급
- **2명씩 묶어서** 등수별 차등 지급

| 등수 묶음 | 지급 장수 |
|-----------|----------|
| 최하위 2명 | 3장 |
| 중위 2명 | 2장 |
| 상위 2명 | 1장 |

- 인원 수에 따라 묶음 기준 자동 조정 (홀수 인원 시 최하위 묶음에 1명 추가)
- 지급 카드는 특수 카드 + 정보 카드 풀에서 **랜덤** 지급

---

### 4.6 라운드 카드

- 매 라운드 결과 시 **항상 1장** 자동 등장
- **전체 회사**에 동시 적용
- 특수 카드 공개 연출 완료 후 마지막에 공개
- **균등 확률**로 랜덤 등장 (총 14장 풀)

| 카드명 | 효과 |
|--------|------|
| 시장 활황 | 전 종목 수익률 +10%p |
| 시장 폭락 | 전 종목 수익률 -10%p |
| 대호재 | 전 종목 수익률 +25%p |
| 대폭락 | 전 종목 수익률 -25%p |
| 변동성 확대 | 전 종목 수익률 절댓값 ×1.5 (방향 유지) |
| 변동성 축소 | 전 종목 수익률 절댓값 ×0.5 (방향 유지) |
| 역전의 날 | 전 종목 수익률 부호 반전 (+ ↔ −) |
| 무풍지대 | 전 종목 수익률 0% 고정 (특수 카드 효과도 무효) |
| 양극화 | 등락 상위 절반 +15%p, 하위 절반 -15%p |
| 버블 | 가장 많이 오른 종목 수익률 추가 ×2 |
| 공황 | 가장 많이 내린 종목 수익률 추가 ×2 (더 급락) |
| 금리 인상 | 전 종목 수익률 -5%p + 보유 현금 +2% 이자 |
| 유동성 장세 | 전 종목 거래량 비례 추가 등락 (많이 산 종목 추가 +, 많이 판 종목 추가 −) |
| 뉴스 블랙아웃 | 이번 라운드 특수 카드 연출 없이 최종 수익률만 표시 |

## 5. 라운드 결과 연출

### 연출 순서

```
① 회사별 특수 카드 공개 애니메이션
     - 모든 회사에 쌓인 카드를 한 장씩 순차 공개
     - 카드 공개 시마다 해당 회사의 수익률 실시간 변동 표시
     - 예: 6개 회사에 [3, 2, 2, 0, 1, 1]장 → 총 9번 애니메이션
     - 뉴스 블랙아웃 라운드 카드 등장 시 이 단계 스킵

② 라운드 카드 공개
     - 특수 카드 전부 공개 후 등장
     - 전체 회사 일괄 적용 연출

③ 최종 수익률 확정
     - 모든 효과 반영 후 회사별 최종 등락 표시 (▲/▼)

④ 순위 발표
     - 등수만 공개 (자산 총액 비공개)
```

### 연출 흐름 예시

```
[회사 A] 급등 카드 공개 → 기본 +15% → +35%
[회사 B] 급락 카드 공개 → 기본 -10% → -30%
[회사 A] 폭등 카드 공개 → +35% → +70%
[회사 C] 반전 카드 공개 → +8% → -8%
         ...모든 특수 카드 소진...
[라운드 카드] "역전의 날" → 전 종목 부호 반전!
[최종 수익률] 회사 A: ▼-70%  회사 B: ▲+30%  회사 C: ▲+8% ...
[순위 발표]   1위 홍길동 / 2위 김철수 / ...
```

## 6. UI 구성

### 6.1 게임 로비 (방 대기실)

- 방 생성 / 입장 (코드 공유 방식)
- 플레이어 목록 및 준비 상태 표시
- 방장 설정 항목: 라운드 수, 라운드 시간, 회사 수 모드(자동/수동)
- 전원 준비 완료 시 방장이 게임 시작

### 6.2 메인 게임 화면 (매매 페이즈)

```
┌─────────────────────────────────────────────────────┐
│  [라운드 표시]  [타이머]        [내 보유 카드 목록] │
├───────────────────────┬─────────────────────────────┤
│                       │  종목 패널                  │
│   주가 차트 영역       │  ┌─────┬──────┬─────┬─────┐ │
│  (선택 종목 차트)      │  │종목명│ 현재가│ 보유│ 매매│ │
│                       │  └─────┴──────┴─────┴─────┘ │
├───────────────────────┴─────────────────────────────┤
│  매매 입력: [종목 선택] [수량] [매수] [매도]          │
│  특수 카드: [카드1] [카드2] ...                      │
│  정보 카드: [카드A] [카드B] ...                      │
│  드래프트:  [선택지1] [선택지2] ← 2라운드부터 표시   │
├─────────────────────────────────────────────────────┤
│  [Tab 홀드 시] 순위표 오버레이                        │
└─────────────────────────────────────────────────────┘
```

**순위표 오버레이 (Tab 홀드 시)**
```
┌──────────────────────┐
│  📊 현재 순위         │
│  1위  홍길동          │
│  2위  김철수          │
│  3위  나              │  ← 본인 강조 표시
│  4위  이영희          │
└──────────────────────┘
```

### 6.3 라운드 결과 화면

- 특수 카드 순차 공개 애니메이션
- 라운드 카드 공개 및 일괄 적용 연출
- 최종 등락률 표시 (▲/▼)
- 등수 발표 (자산액 비공개)

### 6.4 최종 결과 화면

- 최종 순위 + **자산 총액 공개** (종료 후에만)
- 전체 라운드 차트 히스토리 공개
- 플레이어별 수익률 통계
- 다시 하기 / 로비로 버튼

## 7. 게임 흐름

```
[로비] → 방 설정 → 전원 준비 → 게임 시작
    ↓
[게임 시작] 시나리오 생성 + 특수 카드 8장 동일 지급
    ↓
[1라운드] 드래프트 없음 → 바로 매매 페이즈 (3분)
    ↓
[2라운드~] 드래프트 + 매매 동시 진행 (3분)
         → 라운드 결과 연출
         → 등수 발표
    ↓
[3 / 6 / 9라운드 후] 등수 기반 보너스 카드 지급
    ↓
[10라운드 반복]
    ↓
[최종 결과] 순위 + 자산액 + 전체 차트 공개
```
