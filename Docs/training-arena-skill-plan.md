## 1. 개요

### 1.1 목적
- 캐릭터 전투의 **변수**를 만드는 핵심 요소
- 같은 스탯이어도 스킬 구성에 따라 승패가 갈리도록 설계
- 플레이어의 장기 성장(매 회차 스킬 1개 습득)을 뒷받침

### 1.2 기본 규칙
- 전체 스킬 풀: **60종**
- 캐릭터 기본 보유: **3개** (**게임 시작 시 완전 랜덤 배분**, 새 게임 전까지 변경 없음)
- 플레이어도 시작 시 3개 (랜덤)
- 대회 우승자의 스킬 중 **1개를 플레이어가 선택 습득** → 누적
- **플레이어 스킬 슬롯 상한: 8개** — 초과 시 기존 스킬 1개를 새 스킬로 **교체** 선택

### 1.3 스킬 발동 조건
매 턴 행동 시점에 다음 순서로 체크:

```
1. 마나 충분한가? → No면 일반 공격
2. 쿨타임 완료? → No면 일반 공격
3. 적극성 체크 (확률 판정) → 실패 시 일반 공격
4. 사용 가능한 스킬 중 가중치 랜덤 선택 → 발동
```

## 2. 스킬 속성

### 2.1 공통 필드

| 필드 | 설명 |
|------|------|
| id | 고유 ID (예: `skill_001`) |
| name | 스킬명 |
| tier | 등급 (일반 / 희귀 / 영웅 / 전설) |
| category | 분류 (공격 / 방어 / 버프 / 디버프 / 치유 / 특수) |
| target | 대상 (자신 / 단일 적 / 전체 적 / 아군 / 전체) |
| cost | 마나 소모량 |
| cooldown | 쿨타임 (턴 수) |
| priority | 발동 우선순위 (낮을수록 먼저) |
| trigger | 발동 조건 (턴 시작 / HP% 이하 / 피격 시 등) |
| effect | 구체 효과 (스탯 공식 기반) |
| description | 플레이어에게 보여주는 설명 |

### 2.2 등급별 기대 성능

| 등급 | 출현율 (기본 배분) | 기대 효과 |
|------|------|------|
| 일반 | 60% | 단일 타겟, 낮은 배율, 짧은 쿨타임 |
| 희귀 | 25% | 부가 효과 포함 (중독/둔화 등) |
| 영웅 | 12% | 광역 또는 강력 단일 + 조건부 버프 |
| 전설 | 3% | 판 뒤집기급 효과, 긴 쿨타임 |

## 4. 전투 수치 공식 (스킬 연동)

### 4.1 데미지 공식
```
최종 데미지 = (공격력 × 배율 - 방어력 × 계수) × 크리 보정 × 속성 보정
- 크리 발동: 데미지 × 1.5 (행운 기반 확률)
- 방어 무시 옵션: 방어력 × (1 - 무시율) 로 계산
- 최소 데미지: 1
```

### 4.2 치유 공식
```
회복량 = 체력 × 배율
- 최대 HP 초과분은 버림
- 전투 외 회복(단계 이행 등)은 별도 규칙 (상위 기획서 §6.2)
```

### 4.3 지속 효과 (DoT / HoT)
```
턴 종료 시점 일괄 처리
중독/출혈: 턴당 피해
재생: 턴당 회복
```

### 4.4 적극성 → 스킬 발동 확률
```
기본 발동률 = 50% + (적극성 × 0.5%)
- 적극성 100 → 100% 발동 시도
- 마나/쿨타임 모두 충족 + 확률 판정 통과 시 스킬 선택
```

## 5. 스킬 배분 규칙

### 5.1 캐릭터 초기 배분 (확정)
- **완전 랜덤**: 60종 풀에서 중복 없이 3개 추첨
- 아키타입 가중치 없음 (단순화)
- 배분 시점: **게임 시작 시 1회** (슬롯 생성 시)
- 배분 결과는 **새 게임(슬롯 초기화) 전까지 영구 고정** — 매 회차마다 재배분되지 않음

### 5.2 플레이어 스킬 습득 (확정)
- 대회 종료 시 **우승자 + 토너먼트 진출자의 스킬 풀**에서
- 플레이어가 **1개 선택** 습득
- 이미 보유한 스킬은 제외
- **최대 보유 스킬 수: 8개**
  - 보유 8개 미만 → 새 스킬 추가
  - 보유 8개 도달 → 새 스킬을 추가하려면 기존 1개를 선택해 **교체** (습득 포기 옵션도 제공)

### 5.3 전투 중 스킬 선택
```
턴 시작
  → 발동 조건 만족하는 스킬 목록 추출
  → priority 낮은 것 우선
  → 동일 priority면 가중치(등급 + 남은 쿨타임 짧은 순) 랜덤
```
