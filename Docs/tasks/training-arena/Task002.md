# Task002 - 직군별 전술카드 64장 개편

**일자**: 2026-05-01

## 요약

기존 공용 전술카드 8장을 제거하고, 8개 직군마다 전용 전술카드 8장을 제공하는 64장 구조로 개편했다.
매치 프리뷰에서는 현재 플레이어 캐릭터의 직군에 맞는 8장만 노출한다.

## 설계 원칙

- 각 직군은 8장의 선택지를 가진다.
- 각 카드는 최소 2개 상대 직군을 상정해 설계한다.
- 효과 축은 기존 전투 엔진과 호환되는 8종으로 유지한다.
- 카드 이름과 설명은 직군 정체성을 살리고, 실제 수치는 카드 데이터의 `effect`에서 조정한다.

## 효과 축

| 효과 | 전투 적용 |
| --- | --- |
| 선점 | 시작 ATB 증가 |
| 방벽 | 첫 피격 피해 감소 |
| 기습 | 첫 공격 확정 치명 및 피해 증가 |
| 마나 폭주 | 초기 마나 및 마나 회복 증가 |
| 저주 | 상대 ATK/DEF 감소 |
| 회복 | HP 30% 이하 시 1회 회복 |
| 간파 | 상대 회피 0, 내 치명 증가 |
| 최후 | HP 20% 이하에서 ATK/SPD 강화 |
